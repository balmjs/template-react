# BalmJS template - `react`

> A React project

## Requirements

```sh
npm install -g balm-cli balm-core
# OR
yarn global add balm-cli balm-core
```

```sh
# Verify
balm -V

# Output:
# balm-cli: 4.0.0
# balm-core: 4.0.0
```

## Installation

```sh
balm init react my-project

cd my-project
yarn
# or
npm i
```

## Enjoy

```sh
# For development
npm run dev

# For production
npm run prod
```
